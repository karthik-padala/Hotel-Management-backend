A Hotel Management Backend Application -- Airbnb Clone

1. This backend project enables hotel listing, booking, and management with secure JWT-based authentication. 
2. Integrated Stripe Payment Gateway to handle real-time hotel booking payments and confirmations. 
3. Tech Stack: Java, Spring Boot, Spring Security (JWT), MySQL, Stripe API, Lombok 


           Key Backend Features
          -------------------------
Role-based access for Admin, Host, and User 
REST APIs for hotel search, booking, and management 
Secure JWT authentication authorization 
Stripe integration for payment handling and transaction tracking 
Booking confirmation triggered after successful payment
